## 如何使用

您可以通过以下方式使用本仓库中的表情包：

1. **浏览表情包列表**：在本仓库中查看表情包列表，选择您喜欢的表情。
2. **复制表情链接**：复制您喜欢的表情的链接地址，并在需要的地方粘贴使用。
3. **引导他人使用**：将本仓库链接分享给朋友，让更多人分享表情的乐趣。

您也可以在twikoo中直接调用owo.json进行使用：

```
https://fastly.jsdelivr.net/gh/willow-god/owo/owo.json
```
